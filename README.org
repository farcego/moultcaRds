* moultcaRds

** Description

*moultcaRds* is an R package for summarising, analysing and plotting
bird moult card data.

The package is aimed at field moult cards and feather-level moult records.
It provides simple tools to visualise moult extent in wing and tail diagrams.

** Main aims

- Store moult card data in a consistent format than can be easily ported to other related R packages.
- Summarise moult extension by feather tracts andor individual feathers.
- Plot schematic moult extent figures based on Jenni & Winkler book {cite}.
- Prepare moult records for further analysis.
- Keep dependencies minimal (~grid~ es un paquete que se instala junto con R).
