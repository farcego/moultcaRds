# moultcaRds

`moultcaRds` is an R package for summarising, checking, analysing and plotting
bird moult card data.

The package is aimed at field moult cards and feather-level moult records.
It provides simple tools to visualise moult extent in wing and tail diagrams.

## Main aims

- Store moult card data in a consistent format.
- Summarise moult by feather tract.
- Plot schematic moult extent diagrams.
- Prepare moult records for further analysis.
- Keep dependencies minimal.
